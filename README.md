# AI Movie Creator

A multimodal production pipeline using Gemini 3.x to orchestrate storyboards, video clips, and audio assets.

## Project Structure

This workspace is structured as an **AI Film Studio** with a hierarchical inheritance model:

- **Root**: Contains global studio rules and shared skills.
- **Project Silos**: Each project (e.g., `PROJECT_MATRIX`) contains its own lore, aesthetic specifications, scripts, and asset tracking.
